# Typesource

Practice touch typing on source code

# Onboarding

Welcome to typesource, place your cursor on this text and start typing the words you see, the characters will change color while you type them, that means you are typing correctly.

# Debug

Open the debug box on the bottom right to see information about your typing session.

# Github source code

Use the box on the right to search for file on Github repos, pick a file to start practicing on it.
